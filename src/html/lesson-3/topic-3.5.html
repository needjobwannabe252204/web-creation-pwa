🚀 Topic 5: The CSS Box Model

What is the CSS Box Model?

The CSS Box Model is a fundamental concept in CSS that describes the rectangular boxes that are generated for elements in the document tree and how they interact with each other.



Every HTML Element is a Box

In the CSS Box Model, every HTML element is considered a rectangular box.


This box consists of several components:

  • Content: The actual content of the element (text, images, etc.).
  • Padding: The space around the content.
  • Border: The line that surrounds the padding and content.
  • Margin: The space around the border.

    +------------------+
    |     Margin       |
    | +--------------+ |
    | |    Border    | |
    | | +----------+ | |
    | | | Padding  | | |
    | | | Content  | | |
    | | +----------+ | |
    | +--------------+ |
    +------------------+


Example

div {
    padding: 10px;
    border: 2px solid black;
    margin: 15px;
}
