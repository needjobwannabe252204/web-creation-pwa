Topic 1: What is CSS?

CSS stands for Cascading Style Sheets.


While HTML provides the structure of a webpage, CSS controls how it looks.


Think of it this way:


🏠 HTML = House Structure

🎨 CSS = Paint, Decorations, Furniture



Example:


</> CSS
h1 {
    color: blue;
}


Why CSS is Important


Without CSS, websites look plain and difficult to use.


CSS helps:

  • Make websites attractive
  • Improve readability
  • Create professional layouts
  • Make websites work on phones and tablets
  • Improve user experience
