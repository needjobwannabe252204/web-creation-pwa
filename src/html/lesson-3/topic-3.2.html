Topic 2: CSS Syntax

CSS syntax is used to define the style and layout of web pages.

CSS rules are made up of selectors and declarations.


CSS Rule Structure

A CSS rule consists of a selector and a declaration block.


</> CSS 
  
selector {
    property: value;   
}

Example

</> CSS 

h1 {
    color: red;
    font-size: 24px;
}

Selector Description
h1 The first-level heading element.
color A CSS property that sets the text color.
red A color value that sets the text color to red.
font-size A CSS property that sets the size of the font.
24px A length value that sets the font size to 24 pixels.

Important Notes

When writing CSS rules, it's important to remember that:

  • Selectors are used to target HTML elements.
  • Declarations are used to define the styles for the selected elements.
  • Each declaration must end with a semicolon.
