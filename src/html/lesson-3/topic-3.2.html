Topic 2: CSS Syntax


CSS (Cascading Style Sheets) is a language used to control the appearance and layout of web pages. It lets you change colors, fonts, spacing, and positioning of HTML elements.


Syntax

is the set of rules that tells you how to write code correctly in a programming or styling language.



Example:


</> CSS
p {
                    color: red;
                }

"p" -is the selector (chooses all paragraph elements).

"color" -is the property (what to change).

"red" -is the value (the new setting).



correct example


</> CSS
p {
                    color: blue;
                    font-size:  24px;  
                }

incorrect


</> CSS
p {
                    color: blue
                    font-size:  24px  
                }

  • Missing semicolons may cause problems.



common selector


Element selector


</> CSS
p {
                    color: blue
                    
                }

Class selector


</> CSS
.card {
                    background: white
                    
                }

ID selector


</> CSS
#header {
                    background: black
                    
                }


Importance:

  • CSS syntax is important because it tells the browser how to read and apply your styles. If the syntax is incorrect, the browser may ignore some or all of your CSS rules.
