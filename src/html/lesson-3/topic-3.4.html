🚀 Topic 4: Fonts

What are Fonts?

Fonts are a crucial aspect of web design, as they affect readability, user experience, and the overall aesthetic of a website. Choosing the right font can enhance the visual appeal of a site and make content more engaging for users.



Example:

</> CSS 
body {
    font-family: Arial, sans-serif;
}


Font Size

</> CSS 
p {
    font-size: 18px;
}


Font Weight

</> CSS 
h1 {
    font-weight: bold;
}
