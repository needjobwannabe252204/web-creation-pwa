🚀 Topic 4: Fonts

What are Fonts?

Fonts determine how text appears on a webpage.

Fonts are a crucial aspect of web design, as they influence the readability, aesthetics, and overall user experience of a website. In this topic, we will explore the different types of fonts, how to use them effectively, and how to implement them in your web projects.


Example:

body {
    font-family: Arial, sans-serif;
}


Font Size

p {
    font-size: 18px;
}


Font Weight

h1 {
    font-weight: bold;
}
