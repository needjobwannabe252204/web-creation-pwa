📚 Lesson 3

CSS Fundamentals and Responsive Design

Lesson Description

CSS (Cascading Style Sheets) controls the appearance and layout of web pages. In this lesson you will explore CSS syntax, colours, fonts, and the box model. You will also build flexible layouts with Flexbox and create responsive pages that look great on any screen size.

🎯 Learning Objectives

By the end of this lesson, learners will be able to:

  • Explain the purpose of CSS in web development.
  • Apply CSS styles to HTML elements.
  • Use colours and fonts to improve webpage design.
  • Understand and apply the CSS Box Model.
  • Create layouts using Flexbox.
  • Design responsive webpages for different devices.
