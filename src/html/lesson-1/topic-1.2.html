🌐 Topic 2: Web Browsers

What are Web Browsers?

Developers use browsers to check whether their websites work correctly.



Examples of web browsers


  • Google Chrome – The most widely used web browser.
  • Mozilla Firefox – Known for its privacy features.
  • Microsoft Edge – A modern browser built on Chromium.


Why are Browsers Important?

Browsers help developers:

  • Preview their websites.
  • Test functionality.
  • Identify layout problems.
  • Ensure users have a good experience.
