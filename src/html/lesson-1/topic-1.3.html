🎨 Topic 3: Design Tools

What are Design Tools?

Design tools help developers and designers plan how a website will look before coding begins.
They are used to create:

  • Wireframes
  • Layouts
  • Prototypes
  • User interface designs

Why are Design Tools Important?

They help teams:

  • Visualize ideas before coding.
  • Improve user experience.
  • Save development time.
