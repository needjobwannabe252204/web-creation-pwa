Topic 1: HTML Structure

What is HTML?

HTML stands for HyperText Markup Language. It provides the structure and content of a webpage. Think of HTML as the skeleton of a website . Without HTML, websites would have no text, images, buttons, or organization.



Basic HTML Structure


<!DOCTYPE html>  
<html>    
    <head>    
        <title>My First Webpage</title>  
    </head>   

    <body>    
        <h1>Hello World!</h1>   
    </body>   
</html>


Tag Purpose
<!DOCTYPE html> Tells the browser this is an HTML file document
<html> Root element of the webpage
<head> Contains information about the webpage
<title> Displays the title in the browser tab
<body> Contains visible webpage content
