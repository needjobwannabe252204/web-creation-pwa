🔗 Topic 3: Links

Basic hyperlink usage

Links allow users to navigate to other webpages or websites. They use the anchor tag <a>.

Use relative URLs for pages within the same site and absolute URLs for external sites. Make link text descriptive (for example, use "View lesson" instead of "click here") to help accessibility and clarity.



🏷️ Basic Syntax of a Link

Tag / Modifier Purpose
<a></a> anchor tag for hyperlink
href="" An attribute used inside the hyperlink to tell where to go
target="" An attribute used to tell if the link opens a new tab or replace the current tab

Example

<a href="https://www.google.com" target="_blank">google</a>
