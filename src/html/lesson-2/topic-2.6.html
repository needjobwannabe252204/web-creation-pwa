🧇 Topic 6: Tables

What are Tables?

Tables display information in rows and columns.



Example

<table border="1">

<tr>
    <th>Name</th>
    <th>Course</th>
</tr>

<tr>
    <td>Maria</td>
    <td>BTLED-ICT</td>
</tr>

</table>

Result:

Tip: Use tables only for tabular data (rows and columns). For layout and positioning, prefer CSS.


Name Course
Maria BTLED-ICT
