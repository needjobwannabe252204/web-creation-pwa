Topic 4:🖼️ Images

What are Images?

Images add visual content to webpages.

The <img> tag displays pictures.

Always include a meaningful alt attribute for accessibility and choose appropriate image formats (PNG, JPEG, or WebP) and sizes for performance.



Important Attributes

Attribute Purpose
src Location of the image in the file
alt Alternative text describing the image


Example


<img src="cat.jpg" alt="Cute Cat">
