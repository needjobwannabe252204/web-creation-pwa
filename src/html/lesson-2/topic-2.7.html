🗋 Topic 7: Forms

What are Forms?

Forms collect information from users.

Examples include login forms, registration forms, and surveys.



Example

<form>

    <input type="text" placeholder="Enter your name"> 

    <button type="submit">
        Submit
    </button>

</form>


Common Form Elements

Element Purpose
<input> Receives user input
<button> Submits information
<label> Describes an input field
<textarea> Allows longer responses
