🗋 Topic 7: Forms

What are Forms?

Forms collect information from users.

Examples include login forms, registration forms, and surveys.



Example

<form>

    <label for="username">Username:</label>
    <input type="text">
    <button type="submit">

</form>


Common Form Elements

Element Purpose
<input> Receives user input
<button> Submits information
<label> Describes an input field
<textarea> Allows longer responses

Tip: Always pair inputs with <label for="..."> for accessibility. Include the form's method and action attributes when submitting data to a server.
