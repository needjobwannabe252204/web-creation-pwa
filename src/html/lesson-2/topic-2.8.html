Topic 8: Semantic HTML

What is Semantic HTML?

Semantic HTML uses meaningful tags that describe the purpose of webpage sections.

Using semantic tags helps screen readers and search engines understand the structure of your page, improving accessibility and SEO.


It improves:

  • Organization
  • Accessibility
  • Search engine optimization (SEO)


Common Semantic Tags

Tag Purpose
<header> Top section
<nav> Navigation links
<main> Main content
<section> Content section
<article> Independent content
<footer> Bottom section 

<header>
    <h1>My Website</h1>
</header>

<main>

<section>
    <p>Welcome to my webpage.</p>   
</section>

</main>

<footer>
    <p>Copyright 2026</p>
</footer>
